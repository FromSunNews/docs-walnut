# Register as a Worker on Walnet

## Step-by-Step Registration Guide

### 1. Connect New Worker

Choose Register Worker on Worker Workspace. Click **Connect New Worker**

![Connect New Worker](/assets/docs/v1.0.1/provider/2.1.png)

### 2. Enter Worker Name

Enter a unique name for your worker device

![Enter Worker Name](/assets/docs/v1.0.1/provider/2.2.png)

### 3. Select Operating System

Choose the operating system your device is running

![Select Operating System](/assets/docs/v1.0.1/provider/2.3.png)

### 4. Specify Device Type

Specify whether your device is CPU or GPU

![Specify Device Type](/assets/docs/v1.0.1/provider/2.4.png)

### 5. Run Setup Script

Run the script on your devices to set up the worker

![Run Setup Script](/assets/docs/v1.0.1/provider/2.5.png)

### 6. Authorize Device

Complete the authorization process for your device

![Authorize Device](/assets/docs/v1.0.1/provider/2.6.png)
