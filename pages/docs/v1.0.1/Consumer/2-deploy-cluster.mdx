# Deploy Ray Cluster

This guide explains how to deploy a Ray cluster using Walnet's decentralized computing network.

## Overview

Ray clusters on Walnet provide distributed computing capabilities with high-performance GPUs and CPUs. The deployment process is streamlined through our intuitive interface.

## Deployment Steps

### 1. Create Cluster

Select **Deploy Cluster**

![Create Cluster](/assets/docs/v1.0.1/consumer/2.1.png)

### 2. Select Cluster Type

Select the cluster type that meets the scope of your project.

- **General** - This option works well for prototyping or general E2E workloads.
- **Inference** - Choose if you require production-ready clusters for low-latency inference and heavy workloads.
- **Train** - Choose if you require production-ready clusters for machine learning models, training, and fine-tuning.

![Select Cluster Type](/assets/docs/v1.0.1/consumer/2.2.png)

### 3. Choose Location

Please choose the location for GPUs. You can search for a location using the search bar. Ensure that you select the correct location or one close to your residence.

![Choose Location](/assets/docs/v1.0.1/consumer/2.3.png)

### 4. Select Cluster Processor

Please select the processing type for your cluster node. Ensure you choose a configuration that aligns with your specific requirements.

![Select Cluster Processor](/assets/docs/v1.0.1/consumer/2.4.png)

### 5. Summary

In summary, please review the selected configuration carefully, and then click **Deploy Cluster**.

![Summary](/assets/docs/v1.0.1/consumer/2.5.png)
