# Getting Started

Walnet is a decentralized computing platform that revolutionizes how computing resources are shared and utilized. Our network enables users to contribute their unused computing resources to power AI training, blockchain applications, and general computing tasks while earning rewards.

## Use Cases

### 1. AI Model Training

- Distributed machine learning
- Model optimization
- Data processing

### 2. General Computing

- Scientific calculations
- Data processing
- Rendering tasks

### 3. Blockchain Applications

- DApp hosting
- Node operations
